# Projecto-VideoJuegos
Un juego que combina la intensidad de un shooter tactico con la visibilidad estrategica de un MOBA. El jugador controla a un personaje en un entorno tridimiensional visto desde una camara superior. El objetivo es la supervivencia y el cumplimiento de misiones mediante el uso de armamento, habilidades y el aprovechamiento del entorno.
